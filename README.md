# Simone GM Panel Core

Core host and public feature registry for the modular Simone GM Panel.

## Requirements

- Simone Homebrew (`simone-homebrew`) must be installed and active.

## Features

- Hosts the GM-only ApplicationV2 panel.
- Provides the versioned feature registry API.
- Persists window position, collapsed state, and active feature.

## Installation

Install from the Foundry VTT **Add-on Modules** screen using the manifest URL from the latest GitHub release.

## Compatibility

- Foundry Virtual Tabletop 14
- Version: 1.0.0

## Support

Report reproducible issues through this repository's GitHub Issues page. Include the Foundry version, active system version, enabled GM Panel packages, and relevant console errors.
